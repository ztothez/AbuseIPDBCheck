# AbuseIPDB Check Extension

AbuseIPDB Check is a lightweight, open-source browser extension for quickly looking up IP addresses on AbuseIPDB to assess whether they have been reported for malicious activity. It is intended for network administrators, security analysts, and anyone working with potentially untrusted IP addresses.

## Features
* **Right-click lookup**
  Highlight an IP address on any webpage and check it directly via the context menu.

* **Abuse reputation context**
  Opens the AbuseIPDB report page showing confidence score, reports, and abuse history.

* **Privacy-focused design**
  Operates entirely in the browser. No IP addresses are collected, stored, or transmitted by the extension itself.

## Usage
1. Install the extension.
2. Highlight an IP address on a webpage.
3. Right-click and select **Check on AbuseIPDB**.
4. Review the AbuseIPDB report in a new browser tab.

## Permissions
* **Context menus** – required to enable right-click lookups
* **Tabs** – required to open AbuseIPDB result pages

No background requests, tracking, or telemetry are performed.
